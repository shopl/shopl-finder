개인정보처리방침

SHOPL 페이지 분석기는 사용자의 개인 정보를 수집하거나 저장하지 않습니다.

우리의 확장 프로그램은 shoplworks.com , https://www.hadaworks.com/ 관련 페이지에서 1) JSP, React 기술 스택을 구분 2) 디자인시스템 적용 여부를 시각화하는 기능 을 제공하며, 이를 위해 다음과 같은 권한을 사용합니다:

  • activeTab: 현재 활성 탭과 상호작용하기 위해 사용됩니다.
  • scripting: 특정 웹페이지에 기능을 추가하기 위해 사용됩니다.
  • 특정 호스트 권한: shoplworks.com, hadaworks.com 관련 페이지에서만 작동하기 위해 사용됩니다.

이 권한들은 오직 확장 프로그램의 핵심 기능을 제공하기 위해서만 사용되며, 어떠한 개인 정보도 수집하거나 저장하지 않습니다.
